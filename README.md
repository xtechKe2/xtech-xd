## 📋 Deployment

### 1. Fork & Clone
```bash
git clone https://github.com/xtechKe2/xtech-xd.git
cd xtech-xd
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Get Session ID
👉 Click [here](https://t.me/xtechxdbot/) to get your session ID from our Telegram bot.

### 4. Set Environment Variable
Create a `.env` file:
```
SESSION_ID=xtech-md2026;YOUR_SESSION_ID_HERE
PORT=10000
```

Or enter the session ID when prompted on startup.

### 5. Start the Bot
```bash
npm start
```

## 🔧 Configuration

| Setting | Command | Default |
|---------|---------|---------|
| Prefix | `.setprefix` | `.` |
| Mode | `.mode public/private` | `public` |
| Bot Name | `.setbotname` | `XTECH_KE` |
| Owner Name | `.setownername` | `XTECH_KE` |
| Owner Number | `.setownernumber` | - |
| Timezone | `.settimezone` | `Africa/Nairobi` |
| Auto Read | `.autoread on/off` | `off` |
| Always Online | `.alwaysonline on/off` | `off` |
| Chatbot | `.chatbot on/off` | `off` |
| Menu Style | `.setmenu premium/normal` | `premium` |
